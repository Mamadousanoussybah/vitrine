# Portfolio de Mamadou Sanoussybah

Bienvenue sur mon portfolio en ligne !  
Ce site présente mes projets réalisés dans le cadre de mes études et de mes activités personnelles, ainsi que mes compétences techniques en cybersécurité et développement web.

## À propos de moi

Je suis étudiant en **cybersécurité** à l'université du quebec a chicoutimi et passionné par la sécurité informatique, le développement web et la programmation.  
Ce portfolio reflète mon savoir-faire en **HTML, CSS, JavaScript**, ainsi que ma capacité à organiser et structurer des projets.

## Technologies utilisées

- HTML 
- CSS 
- JavaScript  
- Bonnes pratiques de sécurité web (ex. sécurisation des formulaires, protection des données)

## Contenu du portfolio

- Présentation personnelle avec photo  
- Liste détaillée de mes projets réalisés  
- Liens vers les dépôts GitHub de chaque projet  
- Démonstrations en ligne lorsque disponibles  
- Informations pour me contacter

## Objectif du portfolio

Ce portfolio a pour but de mettre en valeur mes compétences en cybersécurité et développement, ainsi que mes réalisations.  
Il est conçu pour faciliter la découverte de mon travail par les recruteurs, professeurs ou collaborateurs.

## Organisation du dépôt

- Le dossier `images/` contient les images utilisées sur le site  
- Le fichier `index.html` est la page d’accueil  
- Les fichiers `style.css` et `script.js` contiennent respectivement les styles et les fonctionnalités JavaScript du site
